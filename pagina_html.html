O conteúdo escrito e sua semântica

<p> e </p> faz parágrafo

<hr> faz uma linha

<br> quebra linha dentro dos parágrafos

Use a tag <b></b> para deixar uma palavra em negrito, mas esta tag não é semântica, foca apenas na forma

Use <strong></strong> para deixar em negrito, a tag <strong></strong> é usada de forma semântica, usada para dar um significado a mais ao texto

Use <i></i> para deixar um termo em itálico, mas é a forma não semântica, focada apenas na forma

Use <em></em> para deixar um termo em itálico, a tag <em></em> é usada de forma semântica, usada para dar ênfase ao texto

Para fazer um texto marcado, use <mark></mark>

Você pode até mudar a cor da marcação, mas é necessário usar CSS (veja o código para ver como é o parâmetro)

Texto marcado

<small></small> é uma tag para deixar as letras pequenas, porém é uma tag semântica pois há nos contratos, as famosas "letras miúdas"

Podemos marcar um texto como excluído usando <del></del>, você pode ler, mas não deve considerá-lo

Podemos marcar um texto como inserido usando <ins></ins>, para dar ênfase à ele

Há também a tag <u></u> para sublinhar, mas não é semântico, tem foco na forma

Para sobrescrever, use <sup></sup> (Exemplo: X4+3)

Para subscrever, use <sub></sub> (Exemplo: H2O - fórmula da água)

A tag <code></code> é usada para facilitar a leitura de códigos, pois deixa o texto monoespaçado

Exemplo:document.getElementById('teste')

Porém, a tag <code></code> perde sua indetação, por exemplo, se seu código tem uma quebra de linha, a tag <code></code> irá ignorar a quebra e colocar tudo na mesma linha

Para evitar isso, coloque a tag dentro da tag <pre></pre>. Veja os exemplos a seguir:

Sem a tag <pre></pre>:

if(num+1==2){ printf("A soma tem resultado 2"); }else{ printf("A soma tem resultado diferente de 2"); }

Com a tag <pre></pre>:

			
				if(num+1==2){
					printf("A soma tem resultado 2");
				}else{
					printf("A soma tem resultado diferente de 2");
				}

Sobre imagens

Colocando imagens

Dica: Gimp é um programa gratuito usado para conversão e edição de fotos, baixe

Nem todas as imagens podem ser utilizadas em seus sites devido ao copyright, atente-se à isso

Unsplash e Pexels são ótimos lugares para se encontrar imagens

Use a tag <img src="nome do arquivo da imagem" alt="um nome pra explicar a imagem"> para usar uma imagem

Você pode usar imagens quando elas estão em sub-pastas (uma pasta dentro da pasta com o arquivo html)

A imagem é a mesma, mas se você olhar o código, verá que o caminho é diferente

Você pode adicionar imagens por link também

Imagens dinâmicas

Nem todos os clientes estão pelo computador, eles não podem ver imagens muito grandes, pois atrapalha e o seu site fica pesado

Para resolver este problema, a tag <picture></picture> combinada com as tags <img></img> e <source></source> fazem com que o site automaticamente selecione imagens menores ou maiores para estes clientes

Áudios em HTML

Para áudios, existe a mesma política das imagens, o copyright, sempre tome cuidado com isso, você não quer processado ou algo do tipo né? rs

Um lugar que você pode arranjar algumas músicas e efeitos sonoros é o próprio Youtube

Abra o Youtube, clique no seu perfil no canto superior direito, clique em "Youtube Studio" e depois "Biblioteca de Áudio"

Você também pode pesquisar músicas NCS (No Copyright Songs) no Youtube, porém...

...para uso comercial das músicas, é necessário contatar a gravadora.

Para mais informações, acesse o site oficial da NCS clicando aqui

O comando mais simples é <audio src="Nome do Arquivo"></audio>

Alguns navegadores não possuem compatibilidade com todos os formatos de áudio(MP3 WAV e OGG são 100% compatíveis)

*CUIDADO COM OS ÁUDIOS WAV, SÃO MUITO PESADOS E PODEM PREJUDICAR A NAVEGAÇÃO NO SITE*

Aqui vai outro jeito de colocar áudios, <audio> <source src="Nome do Arquivo" type="Depende do seu tipo de arquivo, dê uma olhada no MidiaTypes"</audio>, pode colocar vários sources

Parâmetros para a tag audio:

  • Controls: faz a barra de áudio aparecer, você vai poder selecionar o momento da música e dar play
  • Autoplay: como o próprio nome já diz, toca a música sem clicar nela
  • Loop: quando a música termina, toca denovo, isso infinitamente
  • Preload:
    1. Metadata: mostra algumas informações e só carrega quando você clica
    2. Auto: carrega o áudio automaticamente
    3. None: só carrega quando você clicar

Vídeos em HTML

E novamente eu estou aqui avisando, tome cuidado com o Copyright, para ter alguns vídeos sem Copyright, acesse Pexels (atente-se ao tamanho dos vídeos)

E também como os áudios, nem todos os formatos de vídeos são compatíveis (os mais utilizados e suportados são MP4/M4V, WEBM e OGV)

Dica: HandBrake é um programa gratuito e OpenSource, usado para conversão de vídeos, parecido com o Gimp só que para vídeos

Vídeos hospedados localmente

Os vídeos são iguais, mas são de diferentes formatos, respectivamente, .M4V, .MKV, .WEBM e .OGV

Vídeos hospedados externamente

Pode adicionar vídeos por links também, usarei um exemplo do Youtube


<iframe width="500" height="315" src="URL aqui" frameborder="0" allow="accelerometer; autoplay; clipboard-write;
encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>

Clique em um vídeo do Youtube e depois em compartilhar, haverá a opção "Incorporar", o próprio Youtube irá escrever um código para você que adicionará o vídeo ao seu site

Ao incorporar vídeos do Youtube no seu site, você não estará utilizando o servidor que seu cliente irá hospedar, e sim, os servidores do Youtube

Esse método é muito recomendado, pois diminuirá os custos do seu cliente
PS:Aqui também tem copyright

Por isso é importante que você tenha um canal no Youtube e produza seus vídeos, para ficar livre do Copyright

Caso você faça uma site o qual possui conteúdo exclusivo, e você quer que apenas pessoas assinantes do seu site possam visualizá-lo, o Vimeo existe pra isso

Nele, também há o Copyright, possui uma melhor qualidade de vídeo, porém o seu CODEC é meio lento, então faça um equilíbrio, há uma maior personalização de configurações no código para incorporar
