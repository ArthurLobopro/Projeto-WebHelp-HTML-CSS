Bem-Vindos à página sobre JavaScript

Script

Para usar JavaScript em sua página, abra a tag <script></script> dentro do <head></head> ou dentro do <body></body>, é dentro dela que ficará o seu código

Comandos

window.alert('Texto'): lança um alerta.

window.confirm('Texto'): pede uma confirmação do usuário.

window.prompt('Texto'): faz uma pergunta para que o usuário responda.

Comentários

// Deixa uma comentário de apenas uma linha.

/* */ Deixa um comentário de várias linhas, o conteúdo do comentário ficará entre eles.

Tudo que você colocar entre comentários NÃO É considerado como código.

Váriaveis e seus Tipos

Há regras para as identificações de váriaveis e aqui estão elas.

    Regras:

  1. Uma variável NÃO pode ter espaço.
  2. Uma variável deve começar com uma letra, $ ou _.
  3. Uma variável NÃO pode começar com número.
  4. Uma variável pode conter símbolos e acentos.
  5. Uma variável NÃO pode ter nome de comandos do JS.
