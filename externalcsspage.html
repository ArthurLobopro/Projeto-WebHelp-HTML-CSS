CSS Externa

A CSS Externa é a que se escreve em um outro arquivo.

Crie esse arquivo .css e coloque todos os códigos de CSS lá.

A CSS funciona com seletores, você escreve o nome da tag que irá modificar, abre chaves e coloca as declarações
PS:não esqueça de fechar as declarações com ; .

  
    body{
      background-color: #A7D1E5;
      font-family: arial;
    }

Como vincular o arquivo .css ao arquivo HTML:

  
    <link rel="stylesheet" href="NomeDoArquivo./public/css/global.css">>
    
    /*Coloque isso dentro do <head></head>*/
