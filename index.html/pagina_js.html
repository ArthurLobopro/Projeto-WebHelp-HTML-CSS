Bem-Vindo(a) à página sobre JavaScript ^^

Script

Primeiramente, para usar JavaScript em sua página, abra a tag <script></script>, é dentro dela que ficará todo os seus comandos JS

Essa tag deve ser aberta dentro do <body></body>

Comandos

window.alert('Texto'): lança um alerta

window.confirm('Texto'): pede uma confirmação

window.prompt('Texto'): uma pergunta que você pode responder

Comentários

// Deixa uma comentário de apenas uma linha

/* */ Deixa um comentário de várias linhas

Tudo que você colocar entre comentários não será considerado como comando

Váriaveis e seus Tipos

Há regras para as identificações de váriaveis: não pode ter espaço; comece com letra, $ ou _; não pode começar com número; pode usar símbolos e acentos; não pode ter nome de comandos do JS
