CSS Interno

É a CSS que se escreve dentro do <head></head>

Você começa utilizado a tag <style></style>, tudo que você escrever dentro dessa tag será considerado como CSS

A CSS funciona com seletores, você escreve o nome da tag que irá modificar, abre chaves e coloca as declarações
PS:não esqueça de fechar as declarações com ;

      
         <style>
         body{
           background-color: #A7D1E5;
           font-family: arial;
         }
         </style>
